<p><ul><li>Pay To Public Key Hash</li><li>Pay To Public Key</li><li>Pay To Script Hash</li></ul></p>
<p><p>Litecoin transactions can be categorized into three main types: Pay to Public Key Hash (P2PKH), Pay to Public Key (P2PK), and Pay to Script Hash (P2SH). P2PKH involves sending funds to a hashed version of a recipient's public key, enhancing security by requiring the owner to reveal their private key to spend the funds. P2PK entails direct payments to a public key, while P2SH allows for more complex transaction scripts, enabling features like multisig wallets, enhancing versatility and security within the network.</p></p>
